Financial Transaction Analysis & Performance Dashboard

Project Overview
This project involved a comprehensive analysis of 1,110 financial transactions totaling $553,851 in volume. The goal was to evaluate performance across four digital currencies (Auric, Electra, Lumina, Zentia) and identify cost-efficiency trends across three transaction types: Payments, Purchases, and Transfers.

Interactive Tableau Dashboard

I developed an interactive dashboard to translate these complex datasets into actionable business insights.(https://public.tableau.com/app/profile/isaac.kuivi/viz/FinancialTransactionsAnalysis-IsaacKuivi/FinancialTransactionsOperationalAnalysis)

Key Business Insights

Top Performing Currency: Lumina emerged as the primary value driver, accounting for the highest transaction volume (307) and the largest total amount ($152,079).


Revenue & Fee Analysis: While transaction counts were relatively balanced, Transfers generated the highest total fees at $9,768, identifying a key area for potential fee structure optimization.


Volume Trends: Analysis of monthly data revealed a peak in activity during July (151 transactions) followed by a significant 27% decline in August (109 transactions), suggesting seasonal volatility.
